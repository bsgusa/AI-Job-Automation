# Lexora AI — Legal Intelligence Website

A full-featured, production-ready marketing website for **Lexora AI**, a world-class legal technology platform delivering end-to-end AI-powered legal solutions globally.

## Live Preview

Open `index.html` in any browser — zero build step required.

## What's Inside

| Section | Description |
|---|---|
| **Hero** | Animated particle network, live contract analysis dashboard mockup, trust signals |
| **Stats Bar** | Animated counters — 2.4M+ docs, 98% satisfaction, 47 jurisdictions, 60% cost reduction |
| **Solutions** | 6 AI products: Contract Intelligence, Litigation, Compliance, Research, IP, M&A |
| **Technology** | LexLLM™ orbital diagram, security certifications, architecture overview |
| **Industries** | 6 verticals: Law Firms, Corporate, Finance, Healthcare, Tech/IP, Government |

## Design Highlights

- Deep space dark theme with purple/teal gradient brand palette
- Particle network canvas animation with node connections
- Glassmorphism cards, surfaces, and UI elements
- Smooth IntersectionObserver scroll-reveal animations
- 3D tilt effect on product cards
- Cursor glow tracking effect
- Typewriter animation on hero badge
- Animated orbital tech diagram
- Fully responsive — mobile-first to widescreen

## Tech Stack

- **Pure HTML5 / CSS3 / Vanilla JS** — zero dependencies, zero build tools
- Google Fonts: Inter + Playfair Display
- CSS custom properties for consistent theming
- Canvas API for the particle system
- IntersectionObserver API for performant scroll animations
